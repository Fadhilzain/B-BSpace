# B-BSpace
Proyek pengembangan website untuk cafe B&amp;BSpace.

# 1. Mulai (inisialisasi) Git di folder komputermu
git init

# 2. Hubungkan folder lokalmu ke repository GitHub milikmu
git remote add origin https://github.com/Fadhilzain/B-BSpace.git

# 3. Pastikan nama branch utama kamu adalah 'main'
git branch -M main

# 4. PENTING: Ambil (pull) file README.md & .gitignore dari GitHub ke komputermu
git pull origin main --allow-unrelated-histories

# 5. Masukkan seluruh file project NetBeans B&BSpace ke dalam daftar siap upload
git add .

# 6. Simpan catatan commit untuk project B&B Space kamu
git commit -m "Add B&B Space NetBeans project structure and initial JSP"

# 7. Dorong (upload) semua file project ke GitHub!
git push -u origin main
